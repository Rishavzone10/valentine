# Valentine Page

This repo contains a single-page HTML valentine "Will you be my Valentine?" interactive page.

How to use
- Open `index.html` in a browser for local testing.

Deploy options
- GitHub Pages: push this repo to GitHub, then enable Pages from the `main` branch (root).
- Netlify: drag-and-drop `index.html` into the Netlify Drop dashboard or connect the repo.
- Vercel: import the repo and deploy (static site).
